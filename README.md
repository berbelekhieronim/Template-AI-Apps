# Template AI Apps

Starter repository for eye-tracking app templates used in the AI course.

## What is in this repository

- `Visual Search/`: working PsychoPy eye-tracking example app
- `x3-120 SDK/`: Tobii SDK files used by the example app
- `PsychoPy/`: local bundled runtime for development only, intentionally excluded from Git

## Repository policy

This repository does **not** track the local `PsychoPy/` folder. Students should install PsychoPy locally or keep a local runtime next to the repo during development.

## Current example

The existing example app lives in `Visual Search/` and can be used as the base for new course templates.

## Next step

Once the repository is published to GitHub, we can add a clean template app folder derived from the visual search example.
